Эмулятор параллельной машины с бесконечными регистрами
Указания к использованию
1. Для работы эмулятора требуется JRE (лучше - JDK). В операционных системах семейства Windows необходимо добавит в переменную окружения Path путь к папке bin JRE или JDK.
2. Запуск эмулятора и взаимодействие с ним выполняется через командную строчку.
3. Для получения списка опций необходимо набрать "java -jar pmwir.jar -h"
4. Набор команд расширен пятью элементами:
	* I(N) - считывание значения с клавиатуры и запись в регистр с номером N
	* O(N) - вывод значения регистра с номером N на экран
	* s(K, N) - запуск процесса с номером K и связывание с ним регистра с номером N. Перед началом работы связанный регистр будет обнулен. После окончания работы процесса в регистр будет занесена единица. Связанный регистр может быть использован для определения окончания работы процесса.
	* G(N) - выполняет захват регистра с номером N. Доступ на запись захваченного регистра предоставляется только процессу, выполнившему захват. Все остальные процессы при попытке записи входят в состояние ожидания.
	* P(N) - команда освобождения регистра. Может быть выполнена только из процесса, захватившего регистр.
5. Процессы нумеруются с единицы в соответствии с их порядком после параметра "-p".
6. Рекомендуемые параметры запуска "java -jar -i -d -r 500 -p <Процесс 1> <Процесс 2> ... <Процесс 3>"
7. В каталоге examples содержится пример использования эмулятора (запуск командой "java -jar pmwir.jar -i -d -r 500 -p examples/pr1.pmwir examples/pr2.pmwir examples/pr3.pmwir").
8. В настоящий момент эмулятор находится на этапе тестирования, поэтому высока вероятность ошибок в работе.
