Многопрофильная модель релаксационной нейронной сети(РНС)

Совместимость: Windows XP\Vista\7
Для работы программы в режиме формирования отчета необходим установленный MS Word.

Необходимые динамические библиотеки
oleaut32.dll
advapi32.dll
user32.dll
kernel32.dll
msiimg32.dll
gdi32.dll
version.dll
comctl32.dll
shell32.dll
comdlg32.dll
winmm32.dll
